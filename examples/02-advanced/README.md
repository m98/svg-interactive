# Advanced Examples

Explore advanced features like custom styling, validation, and state management.

## Examples in This Folder

### 1. Custom Styling (`custom-styling.tsx`)
Tailwind CSS integration and custom themes.

**What you'll learn**:
- Using `theme="none"` for full control
- Tailwind utility classes
- Custom CSS styling
- CSS-in-JS with inline styles

### 2. Custom Components (`custom-components.tsx`)
Replace default inputs/outputs with custom React components.

**What you'll learn**:
- `renderInput` prop
- `renderOutput` prop
- Building custom number inputs
- Building custom output displays

### 3. Controlled State (`controlled-state.tsx`)
External state management with useState.

**What you'll learn**:
- Controlled input values
- Controlled output values
- Managing state externally
- Complex computations

### 4. Validation (`validation.tsx`)
Input validation and error handling.

**What you'll learn**:
- Real-time validation
- Error messages
- Type constraints
- Range validation

## Running These Examples

Copy any example to your project:

```tsx
import { CustomStyling } from './custom-styling';

function App() {
  return <CustomStyling />;
}
```
