## Quick Comparison

| Tool | ID Method | Difficulty | Best For |
|------|-----------|------------|----------|
| draw.io | data-id | Easy | Flowcharts, diagrams |
| Figma | Layer names | Easy | UI designs, mockups |
| Inkscape | Object IDs | Medium | Detailed SVG work |
| Illustrator | Layer names | Medium | Professional graphics |
| Hand-coded | Direct IDs | Hard | Full control, optimization |

## Using These Examples

1. Open the SVG file in your tool of choice
2. Study how IDs or data-ids are structured
3. Export following the tool's guide
4. Load in your React app with matching config
