# Real-World Examples

Complete, production-ready applications.

## Examples in This Folder

### 1. Physics Simulator (`physics-simulator.tsx`)
Gas laws calculator (PV = nRT) with units and validation.

**Features**:
- Multiple units support (SI, Imperial)
- Scientific notation
- Input validation
- Real-time computation
- Educational tooltips

### 2. Financial Calculator (`financial-calculator.tsx`)
Loan payment calculator with amortization.

**Features**:
- Loan amount, rate, term inputs
- Monthly payment output
- Total interest calculation
- Amortization schedule
- Currency formatting

### 3. Data Dashboard (`data-dashboard.tsx`)
Multi-metric analytics dashboard.

**Features**:
- Multiple data inputs
- Computed KPIs
- Progress indicators
- Conditional formatting
- Export functionality

### 4. Process Flow (`process-flow.tsx`)
Workflow automation with status tracking.

**Features**:
- Multi-step process
- Status indicators
- Conditional logic
- Progress tracking
- Validation rules

## Running These Examples

These are complete applications. You can:

1. **Copy to your project**:
   ```bash
   cp examples/05-real-world/physics-simulator.tsx src/
   ```

2. **Use as templates**:
   - Adapt the logic to your use case
   - Modify SVG diagrams
   - Customize styling

3. **Deploy standalone**:
   - Each can run as a standalone app
   - Production-ready code
   - Full error handling
