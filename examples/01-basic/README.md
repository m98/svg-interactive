# Basic Examples

Simple starter examples to understand the core concepts.

## Examples in This Folder

### 1. Hello World (`hello-world.tsx`)
The absolute minimum to get started. Single input, single output.

**What you'll learn**:
- Basic setup
- Simple pattern matching
- Output computation

### 2. Calculator (`calculator.tsx`)
Simple two-number calculator with sum and product.

**What you'll learn**:
- Multiple inputs
- Multiple outputs
- Basic arithmetic operations

### 3. Temperature Converter (`temperature-converter.tsx`)
Convert between Celsius and Fahrenheit.

**What you'll learn**:
- Real-world calculation
- Number formatting
- Input validation basics

## Running These Examples

### In Your React App

1. Copy the example file to your project
2. Import and use:
   ```tsx
   import { HelloWorld } from './hello-world';

   function App() {
     return <HelloWorld />;
   }
   ```

### As Standalone Demo

Each example can run standalone:
```bash
npm run dev
# Open http://localhost:3000
```
